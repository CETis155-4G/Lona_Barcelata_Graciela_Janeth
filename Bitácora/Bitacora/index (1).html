HTML

En esta clase vimos el significado de HTML.

HTML es un lenguaje de etiquetas de hipertexto, es el componente mas basico de la web.

El Hipertexto hace referencia a enlaces que conectan a paginas web entre si, ya sea dentro de un unico sitio web.

HTML utiliza marcas para etiquetar texto,imagenes etc. Incluyen elementos especiales como por ejemplo: head,title,body,header,p,img,audio,canvas,video etc. Un elemento HTML se distingue de otro texto en un documento mediante etiquetas, que consiste en el nombre de un elemento con estos signos "<"y">".

Tambien vimos que no solo existe una version de HTML si no que existen 5 versiones.
